# Fund Catalog Semantic Search POC

Base React application for the frontend proof of concept.

## Stack

- Node 22
- Vite
- React 18
- TypeScript 5
- Redux Toolkit
- Redux-Saga
- Axios
- Sass

## Available scripts

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
npm run preview
```

## Current scope

This repository currently contains only the application foundation:

- Vite-based React app
- typed Redux store
- root saga wiring
- base bootstrap flow
- minimal shell UI

No catalog or semantic-search features have been added yet.
